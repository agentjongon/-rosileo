Vector Image I created the warrior in the 'Paint New Sprite'. It was created using the .svg file format. As .svg files compared to .jpg files do not lose their quality. I shaped his body carefully using the vector style of drawing.
Vector graphics, unlike bitmap or raster graphics which are defined as a set of pixels, instead use code to describe a graphic and my or anybodies computer will then read that code and render a graphic from it.
Logical Operators The use of Logical operators is used to end the game. There is an 'and' and 'or' operator inside of the game. They were used in the bat sprite as it was the item needed to be collected. Logical Operators or Boolean/ Binary are basic operations that direct the flow of computing towards different results. It is used to analyze and simplify the digital/ logic circuits. It uses only the binary numbers 0 and 1. At the centre of Boolean Logic is the idea that all values are either true or false. The use of Boolean Logic allows for the creation of more complex definitions, allowing for coders to build a very specific set of definitions.
Number systems are different ways to represent numbers on computers, different systems are used in different systems. Thus being important to be able to distinguish and convert systems with one another.
